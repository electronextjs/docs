# Customizando o Menu 
![ElectroNext.js Example](/img/app.custom.menu.png)

As opções do menu principal são apenas amostras, você pode criar seu próprio menu usando componentes electronext ou usando seus próprios componentes estilizados, veja como fazer abaixo! 

## Criando Menu
Crie um componente `Menu` na pasta `components`
```js
//map
App
├─> components
│   └─> menu.tsx
├── pages
├── public

```

```tsx
//menu.tsx

import { ButtonList, DrawerHeader, DrawerMain, DrawerFooter, Hr  } from "electronext"

export const Menu = () => {
  return(<>
    // link[6:18] /docs/App/components/DrawerHeader
    <DrawerHeader>
      <ButtonList label="Home"/>
    </DrawerHeader>
    // link[6:16] /docs/App/components/DrawerMain
    <DrawerMain>
      // link[8:10] /docs/App/components/Hr
      <Hr/>
      // link[8:18] /docs/App/components/ButtonList
      <ButtonList label="Team Members"/>
      <ButtonList label="Stats"/>
    </DrawerMain>
    // link[6:18] /docs/App/components/DrawerFooter
    <DrawerFooter>
      <ButtonList label="Settings" icon={'⚙️'}/>
    </DrawerFooter>
  </>)
}
```

## Adicionando Menu
Na pasta `pages` localize o arquivo `_electronext.tsx`
```jsx
//map
App
├─> components
│   └─> menu.ts
├── pages
│   └─> _electronext.tsx
├── public

```
Importe seu componente `Menu` e adicione em `ElectroNextApp` a propriedade `CustomMenuDrawer` com seu valor sendo seu `Menu`
```tsx focus=1,11
//_electronext.tsx

import { Menu } from 'components/menu'

const AppElectroNext = ({children}) => {
  const config = enconfig as ElectroNextAppConfig
  return (
    <ElectroNextApp 
      Electron={globalThis.Electron}  
      config={config}
      CustomMenuDrawer={<Menu/>} 
    >
      {children}
    </ElectroNextApp>
  )
}
```
