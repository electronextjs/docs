# ElectroNext.js

ElectroNext.js é um exemplo de aplicação [Electron.js](https://www.electronjs.org) + [Next.js](https://nextjs.org) usando [Typescript](https://www.typescriptlang.org) com [Vibrancy](https://github.com/Seo-Rii/electron-acrylic-window) para efeito acrílico na janela do aplicativo.

Este exemplo tem alguns componentes úteis para usar e manter a UI do aplicativo padrão, confira a [documentação](https://electronextjs.vercel.app/docs) to help you!  para te ajudar!

Você pode criar um exemplo deste aplicativo usando seu próprio CLI [`create-electronext-app`](https://github.com/electronextjs/create-electronext-app).
